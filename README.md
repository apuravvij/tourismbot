# Tourism Bot

A simple tourism chatbot powered by Mistral AI.

## Setup

1. Ensure you have Python 3.9+ installed.
2. Create and activate a virtual environment (already done).
3. Install dependencies: `pip install -r requirements.txt` (or they are installed).
4. Set your Mistral API key in `.env` file.
5. Run the app: `streamlit run app.py`

## Usage

- Open the Streamlit app in your browser.
- Ask questions about tourism, destinations, travel tips, etc.
- The bot will respond with helpful information.

## Requirements

- mistralai
- streamlit
- python-dotenv
